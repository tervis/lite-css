Spacing

Lite CSS Framework uses a simple spacing system inspired by the likes of BassCSS and Tachyons.
You use t (top), b (bottom), l (left), and r (right) to say which direction you want, and the number at the end is the level.
pl-2 pr-2
In this example, p means padding with levels from 0-5.
ml-1 mt-2
This example shows off margins. For the middle selector, you use t (top), b (bottom), l (left), and r (right) to say which direction you want the margin.
p-2 m-2
If you want something on all sides, just do the above, with type then level.
